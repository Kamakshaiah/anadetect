# anadetect
A module that support outliers detection in univariate and bivariate data sets. 

Open anadetect.R file in RStudio, use CONTROL + ENTER to execute the code. After that, you may use functions in the module. At present this module supports outliers detection at any give CIs like 90%, 95% or ....Read the documentation (.PDF) file for more details.
